# Video Marker
로컬에서 동작하는 비디오 플레이어로 비디오의 특정 시간을 마킹하여 이동할 이동할 수 있습니다.

로컬 환경의 동영상 플레이어는 다양한 형식과 코덱의 제공을 통한 강력한 동영상 재생 기능에 집중한다. 그러나 동영상 공유 웹사이트 등에서 제공하는 타임스탬프 댓글, 재생목록등의 동영상과의 상호작용을 할 수 있는 편의기능은 제공하는 앱은 시중에 없다. 이러한 상호작용 기능을 동영상 플레이어의 장점인 편리한 플레이어 화면 UI와 결합하여 북마크 기능외 편의기능을 제공하는 방식의 앱을 개발하고자 한다.
